## TOP
- 시스템의 상태를 전반적으로 확인하기 위해
- mac, linux의 기본 프로그램으로 설치되어 있음
- 옵션 없이 실행하면 interval 간격을 3초로 화면을 갱신하며 보여줌

## 단축키
- 소문자 o 를 입력하면 어떤 순으로 출력할지를 정한다.
- primary key : CPU (CPU 사용 순으로 출력)
- primary key : MEM (MEM 사용 순으로 출력)
